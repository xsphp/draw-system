# 述职抽签系统 - 使用说明

## 📦 文件位置

**路径**: `/home/admin/openclaw/workspace/draw-system/index.html`

双击即可在浏览器中运行，无需服务器！

---

## ✨ 超简单使用方式

### 1️⃣ 打开网页

双击 `index.html` 文件

### 2️⃣ 输入配置

在配置框中按以下格式输入：

```
【中心名称：抽取次数】
人员姓名 人员姓名 人员姓名

【另一个中心:2】
张三 李四 王五
```

### 3️⃣ 保存配置

点击「💾 保存配置」按钮

### 4️⃣ 开始抽取

点击「🎯 开始抽取」或「⚡ 自动抽完」

---

## 📋 配置格式详解

### 格式说明

```
【中心名称：抽取次数】
人员 1 人员 2 人员 3

【中心名称：抽取次数】
人员 1 人员 2
```

### 实际示例

```
【安全环保中心:1】
李武 苏昌丽 廖贵生

【包装中心:2】
佘雁铬 许祥玲

【生产中心:1】
张三 李四 王五 赵六
```

### 规则说明

| 要素 | 说明 | 示例 |
|------|------|------|
| 中心名称 | 任意文字 | 安全环保中心 |
| 抽取次数 | 数字 | 1、2、3... |
| 人员姓名 | 空格或逗号分隔 | 张三 李四 王五 |
| 分隔 | 空行分隔不同中心 | 换行 + 空行 |

---

## 🎯 抽取规则

1. **第一轮**：随机抽取一个还有名额的中心
2. **第二轮**：在该中心剩余人员中随机抽取 1 人
3. **防重复**：已中签人员不会再次被抽取
4. **自动停止**：当所有中心名额用完或人员抽完时停止

## 🔧 功能按钮

| 按钮 | 功能 |
|------|------|
| 💾 保存配置 | 保存当前配置到浏览器 |
| 📋 加载示例 | 填入示例配置 |
| 🗑️ 清空配置 | 清空配置框 |
| 🎯 开始抽取 | 单次抽取 1 人 |
| ⚡ 自动抽完 | 一次性抽完所有名额 |
| 🔄 重置抽取 | 清空记录，重新抽取 |
| 📤 导出记录 | 导出为 CSV 文件 |
| 🗑️ 清空记录 | 删除所有抽取记录 |

---

## 💾 数据保存

- **配置**：自动保存在浏览器 localStorage
- **抽取记录**：自动保存，关闭浏览器不丢失
- **导出备份**：可导出 CSV 文件永久保存

---

## ⚠️ 注意事项

1. **浏览器兼容**：建议使用 Chrome、Edge、Firefox
2. **配置修改**：修改后需点击「保存配置」
3. **清除缓存**：清除浏览器数据会丢失配置和记录
4. **人员姓名**：确保不重复，否则可能影响统计
